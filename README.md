# Quant Trading Strategy (Python)

## Overview
This project implements a basic quantitative trading strategy using a moving average crossover model. It simulates buy and sell decisions based on price trends and evaluates overall profitability.

## Strategy Type
This is a **momentum-based strategy** using:
- Short-term moving average
- Long-term moving average

## How It Works
- When the short-term average crosses above the long-term average → BUY
- When it crosses below → SELL

## Features
- Signal generation (buy/sell)
- Trade simulation
- Profit calculation
- Basic backtesting

## Technologies
- Python
